
# CRM-система для сети музыкальных школ

Выпускная квалификационная работа: разработка CRM-системы для автоматизации основных бизнес-процессов сети музыкальных школ.

## Технологический стек

- **Бэкенд**: Python 3.12, FastAPI, SQLAlchemy 2.0 (async), Alembic
- **База данных**: PostgreSQL 16
- **Фронтенд**: HTML/CSS/JavaScript (этап MVP, далее планируется React)
- **Веб-сервер фронта**: nginx (с проксированием API)
- **Контейнеризация**: Docker, Docker Compose
- **Менеджер пакетов Python**: uv
